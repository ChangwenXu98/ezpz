Overview

ezpz üçã

Launch and train across all your accelerators, using your favorite framework + backend combo.

ezpz simplifies the process of:

  • Setting up + launching distributed training:
    • import ezpz as ez

      • RANK = ez.setup_torch(backend=backend) for backend \in {DDP, deepspeed, horovod}

      • RANK = ez.get_rank()

      • LOCAL_RANK = ez.get_local_rank()

      • WORLD_SIZE = ez.get_world_size()

      (see ezpz/dist.py for more details).

  • Using your favorite {framework, backend}

    On any accelerator:

    • framework=pytorch + backend={DDP, deepspeed, horovod}

    • framework=tensorflow + backend=horovod

    • ez.get_torch_device(): {cuda, xpu, mps, cpu}

    • ez.get_torch_backend(): {nccl, ccl, gloo}

    2ez üòé. (see frameworks for additional details)

  • Writing device agnostic code:
    ezpz_data_parallel.py 
    """
ezpz_ddp.py

- to launch:

  $ source ezpz/src/ezpz/bin/savejobenv
  $ BACKEND=DDP launch python3 ezpz_ddp.py
"""
import os
import logging
import torch
import ezpz as ez

# backend can be any of DDP, deespepeed, horovod
RANK = ez.setup_torch(
    backend=(
        backend := os.environ.get('BACKEND', 'DDP')
    )
)
WORLD_SIZE = ez.get_world_size()
DEVICE = ez.get_device()

# log only from RANK == 0
logger = logging.getLogger(__name__)
logger.setLevel("INFO") if RANK == 0 else logger.setLevel("CRITICAL")

model = torch.nn.Linear(3, 4)
model.to(DEVICE)
optimizer = torch.optim.Adam(model.parameters())
if WORLD_SIZE > 1:
    if backend.lower() == 'ddp':
        from torch.nn.parallel import DistributedDataParallel as DDP
        model = DDP(model)
    elif backend.lower() in ('ds', 'deepspeed'):
        import deepspeed
        model, optimizer, *_ = deepspeed.initialize(
            model=model,
            optimizer=optimizer
        )

x = torch.tensor([1.0, 2.0, 3.0]).to(DEVICE)
y = model(x)
loss = y.sum()
if backend == 'deepspeed':
    model.backward(loss)
    model.step(loss)
else:
    loss = loss.backward()
    optimizer.step()

Details

We can launch on any of {ThetaGPU, Polaris, Perlmutter}\left(^{\ast}\right) with a specific {framework, backend} combo by

  1. savejobenv:

    $ source src/ezpz/bin/savejobenv

    • This will export launch=<launcher> <launcher-opts> for <launcher> \in {mpirun,mpiexec,srun} on (^{\ast}) respectively.

    • By default, launch <exec> will launch <exec> across all the available GPUs in your active {COBALT,PBS,slurm} job.

  2. launch

    $ launch $(which python3) -m ezpz framework=<framework> backend=<backend>
    • Will launch __main__.py (in this case) with framework <framework> and backend <backend> (e.g. pytorch and deepspeed)
